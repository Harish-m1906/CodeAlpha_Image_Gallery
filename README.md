📸 Image Gallery Project
🚀 About

This is a responsive Image Gallery web application built using HTML, CSS, and JavaScript. 
It features a modern UI with interactive image viewing and category filtering.

✨ Features
Responsive design for all devices
Image grid gallery layout
Category filtering (Nature, Tech, Animals, City)
Lightbox image preview
Next / Previous navigation
Smooth hover effects and animations
Keyboard controls (Arrow keys + ESC)

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)

📂 Project Structure
Image-Gallery/
├── index.html
├── style.css
├── script.js
└── README.md

▶️ How to Run
Clone the repository
Open the folder
Run index.html in any browser
